# terminal-tron

![tron game picture](https://i.ibb.co/chMygjh/image-1.png)

Simple Tron game played in the terminal.

## To play:

Clone repository:

```
git clone https://github.com/IrfanThomson/terminal-tron.git
```

Change directory:

```
cd terminal-tron
```

For single player:

```
python3 tron.py
```

For two player:

```
python3 tron2p.py
```
